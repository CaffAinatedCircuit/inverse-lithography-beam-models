# Computational Lithography via Localized Coherent Beam Synthesis
Overview

This project explores a computational approach to lithography pattern synthesis using coherent optical field models. Traditional interference lithography relies on plane-wave interference to produce periodic patterns such as photonic crystals or gratings. However, such methods are limited in their ability to represent arbitrary asymmetric mask geometries.

In this work, we investigate whether complex mask patterns extracted from GDS layout files can be approximated using a superposition of localized coherent beams. We compare two models:

- *Localized Gaussian-modulated beam model*

- *Ewald-constrained interference model*

The first model allows arbitrary spatial frequencies and localized envelopes, while the second enforces a physical wavelength constraint consistent with classical interference lithography.

The goal is to understand the representational limits of coherent beam synthesis under physical constraints and evaluate reconstruction fidelity across varying beam counts.

# Problem Statement

Let  T(x,y) denote a binary mask pattern extracted from a lithographic layout.

We seek to approximate this pattern using the intensity of a coherent optical field:

![eq](https://latex.codecogs.com/svg.image?I(x,y)=|E(x,y)|^2)
where E(x,y) is the electric field generated by a set of coherent beams.

The central question is:

How accurately can arbitrary mask patterns be represented using a finite number of coherent beams under physically constrained optical models?

GDS Layout Processing

The system reads layout files in GDSII format, which is the standard representation for semiconductor masks.

Each file contains:
- hierarchical cells
- polygon geometries
- layer identifiers
For each GDS file:
- All top-level cells are parsed.
- Unique layer identifiers are detected.
- Each layer is rasterized into a binary mask.

Rasterization converts polygon geometry into a grid representation:
T(x,y)∈{0,1}
### Electric Field Model

The electric field is defined as:

![equation](https://latex.codecogs.com/svg.image?E(x,y)=\sum_{n=1}^{N}c_ne^{i(k_{x,n}x+k_{y,n}y)})
representing exposed and non-exposed regions.

Rasterization Method
- For each polygon:
- Extract vertex coordinates.
- Create a polygon path.
- Test whether each grid point lies inside the polygon.

The result is a dense grid:

resolution × resolution
typically:
256 × 256
which becomes the target mask used for optimization.

# Optical Field Models

Two optical field representations are evaluated.

## Model 1: Localized Gaussian-Modulated Beam Model

The electric field is defined as:

![eq](https://latex.codecogs.com/svg.image?E(x,y)=\sum_{n=1}^{N}c_ne^{i(k_{x,n}x+k_{y,n}y)}\exp\left(-\frac{(x-x_n)^2+(y-y_n)^2}{\sigma_n^2}\right))

This model represents localized wave packets similar to Gabor functions.

### Properties
- arbitrary spatial frequencies
- localized spatial support
- high expressive capacity
- not physically constrained by wavelength

## Ewald-Constrained Interference Model

For physically realizable interference lithography, the spatial frequency must satisfy the Ewald constraint:

<img src="https://latex.codecogs.com/svg.image?k_x%5E2%20%2B%20k_y%5E2%20%3D%20k_0%5E2"/>

where
![eq](https://latex.codecogs.com/svg.image?k_0=\frac{2\pi}{\lambda})

This restricts spatial frequencies to the Ewald circle.
The spatial frequencies can therefore be parameterized using an angle:
<p align="center">
<img src="https://latex.codecogs.com/svg.image?k_x%20%3D%20k_0%5Ccos%5Ctheta%2C%5Cquad%20k_y%20%3D%20k_0%5Csin%5Ctheta"/>
</p>

Substituting this constraint into the field model yields

![eq](https://latex.codecogs.com/svg.image?E(x,y)%20%3D%20%5Csum_%7Bn%3D1%7D%5EN%20c_n%20e%5E%7Bi(k_0(%5Ccos%5Ctheta_n%20x%20%2B%20%5Csin%5Ctheta_n%20y))%7D%20%5Cexp%5Cleft(-%5Cfrac%7B(x-x_n)%5E2%20%2B%20(y-y_n)%5E2%7D%7B%5Csigma_n%5E2%7D%5Cright))

### Properties
- fixed wavelength
- physically realizable interference
- reduced spectral freedom

# Optimization Objective
The model parameters are optimized by minimizing mean squared error:
![eq](https://latex.codecogs.com/svg.image?\mathcal{L}=\frac{1}{M}\sum_{x,y}(I(x,y)-T(x,y))^2)

Optimization uses the Adam optimizer with gradient descent.

# Training Procedure

For each configuration:
- Initialize beam parameters randomly.
- Compute field intensity.
- Evaluate reconstruction error.
- Backpropagate gradients.
- Update parameters.
- Training proceeds for a fixed number of epochs.
- Beam Count Scaling

To study representational capacity, experiments are conducted with varying beam counts:
- 4 beams
- 8 beams
- 16 beams
- 64 beams

Increasing beam count increases model expressiveness but also increases optimization complexity.

# Evaluation Metrics

Multiple metrics are used to evaluate reconstruction quality.

### Mean Squared Error (MSE)
![eq](https://latex.codecogs.com/svg.image?MSE=\frac{1}{N}\sum(I(x,y)-T(x,y))^2)
Measures overall reconstruction accuracy.

### Structural Similarity Index (SSIM)
SSIM evaluates perceptual similarity between images:
![eq](https://latex.codecogs.com/svg.image?SSIM(T,I))
Captures structural differences not reflected in pixel-wise metrics.

### Edge Overlap

Edge overlap compares edge maps of reconstructed and target masks.
Steps:
- Compute Sobel edge map.
- Measure intersection of detected edges.

<p align="center">
<img src="https://latex.codecogs.com/svg.image?EdgeOverlap%20%3D%20%5Cfrac%7B%7CE_T%20%5Ccap%20E_I%7C%7D%7B%7CE_T%7C%7D"/>
</p>

### Fourier Spectral Match

The spectral distribution of the reconstructed pattern is compared with the target.

Steps:
- Compute 2D Fourier transform.
- Compare spectral magnitude distributions.

![eq](https://latex.codecogs.com/svg.image?SpectralMatch=corr(F_T,F_I))
This metric evaluates frequency-domain fidelity.

# Experimental Observations

Initial experiments reveal:
- Localized Gaussian beam models achieve lower reconstruction error due to greater spectral freedom. Also these models excel at places with large features that would have taken fourier series to generate infinite sequences.
Example:

- Ewald-constrained models exhibit higher error because spatial frequencies lie on a fixed ring.
- Increasing beam count improves reconstruction until performance saturates.
- Edge fidelity remains the most challenging aspect due to sharp discontinuities in mask geometry.

## Implications for Lithography

These results suggest:

- Classical interference lithography is fundamentally limited in representing arbitrary layouts.
- Localized coherent beam synthesis significantly improves representational capacity.
- Physical wavelength constraints impose spectral limitations that restrict achievable mask geometries.
- Understanding these limits may inform new lithographic techniques using programmable optical fields.
